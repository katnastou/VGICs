1.humsavar.txt -> το αρχειο που διαχειριστηκαμε απο τον FTP εξυπηρετητη της uniprot. Περιεχει πληροφορια για τα missense snps του ανθρωπου
(version:18-07-2018)

2.uniprot_variants.pl -> script με το οποιο απομονωνουμε τα snps που αφορουν σε VGICs του ανθρωπου. Σαν output λαμβανουμε:
α)το ονομα του γονιδιου
β)το uniprot ac
γ)το φυσιολογικο αα με τον κωδικο του 1 γραμματος
δ)τη θεση του πολυμορφισμου
ε)το snp με κωδικο ενος γραμματος
στ)τον κλινικο χαρακτηρισμο του πολυμορφισμου

3.uniprot_ds.txt -> αρχειο output που προκυπτει απο το script 2.  Στην ουσια 
ειναι το συνολο δεδομενων των missense snps για τα VGICs του ανθρωπου που
συλλεχθηκε απο τη βαση UniProtKB

Πορεια εργασιας για τη ληψη των bootstrapped samples σε γλωσσα R στο περιβαλλον 
του R-studio:(βηματα)Για τη διεξαγωγη ολης της παρακατω διαδικασιας χρειαζεται να ειναι εγκαταστημενες και ενεργοποιημενες οι βιβλιοθηκες 
plyr, dplyr και boot.

1. παρσαρεις το φυλλο του excel που σε ενδιαφερει (στη δικη μας περιπτωση το excel ειναι το 
aa_frequencies και τα φυλλα εργασιας ειναι:
α)only_polymorphisms
b)only_pathogenic
c)PORE
d)VSD

2.κατασκευαζεις υποσυνολο που να εχει σε μια στηλη την αμινοξικη αλλαγη
(πχ ΑD, A φυσιολογικο αμινοξυ και D το snp). Για την κατασκευη του υποσυνολου χρειαζονται 
 τα παρακατω βηματα με την εξης σειρα:
i)θετεις υποσυνολο: (παρατειθεται παραδειγμα)
name1 <- subset(aa_frequencies, επιθυμητη στηλη με clinical sig(π.χ X__6) == "Pathogenic")

ii)φτιαχνεις τη μορφοποιηση του υποσυνολου, ωστε να εχεις σε μια στηλη φυσιολογικο αα και snp,οπως ειπαμε παραπανω.
Οποτε ΕΠΙΛΕΓΕΙΣ εκεινες τις στηλες που εχουν πληροφορια για ΦΥΣΙΟΛΟΓΙΚΟ ΑΑ ΚΑΙ SNP:
name2 <- paste(name1$ονομα στηλης, name1$ονομα στηλης)

iii)δινεις μορφη data.frame στο υποσυνολο:
name2 <- data.frame(name2)
colnames(name2) <- c("aatosnp")

3.θετεις το απαραιτητο function για την εντολη boot που θα χρησιμοποιησεις μετα
(εδω εμεις κανουμε count την εμφανιση καθε αλλαγης):
my.counts <- function(name2,d){return(count(name2[d]))}
Στην ουσια στην πρωτη παρενθεση θετεις το συνολο που θα εφαρμοστει το function και το
δεικτη (εδω ονομαζεται d) που δηλωνει τη στηλη του συνολου με την οποια θα δουλεψει.
Επειτα, δηλωνεται η εντολη που θελεις να τρεξει το function (εδω η εντολη ειναι το count που μετραει
τη συχνοτητα εμφανισης καθε αα-αλλαγης)

4.φτιαχνεις τα θεωρητικα συνολα (με τη μεθοδο bootstrap) που χρειαζεσαι για την εφαρμογη της μεθοδου odds ratio:
my.replicates <- replicate(1000, boot(name2$aatosnp, my.counts, 1))
Εδω περα με την εντολη replicate θα φτιαξεις τα 1000 θεωρητικα συνολα, ενω με την εντολη boot
εφαρμοζεις τη μεθοδο bootstrap στο αρχικο σου συνολο). Το αποτελεσμα ειναι ενας πολυδιαστατος πινακας
απο τον οποιο στη συνεχεια απομονωνουμε συγκεκριμενες γραμμες

5.Απομονωνουμε τις γραμμες που περιεχουν τα θεωρητικα συνολα. Βρισκονται στη γραμμη με ονομα "t". Τις παρσαρουμε
σε πολυδιαστατο πινακα
my.lines <- data.frame(t(my.replicates["t",]))
Η εντολη t() κανει αντιμεταθεση των γραμμων σε στηλες

6.εξαγουμε ολα τα θεωρητικα συνολα με μορφη αρχειων κειμενου για να τα διαχειριστουμε
στη συνεχεια με προγραμμα σε γλωσσα perl με το οποιο θα υπολογισουμε το θεωρητικο μεσο ορο
και την τυπικη αποκλιση σε καθε θεωρητικο συνολο:
for (i in 1:1000){
write.table(my.lines[[i]], paste ("filename", i, "txt", sep = "."), sep = "\t")
}
Οποτε λαμβανεις 100 αρχεια κειμενου,καθε ενα εκ των οποιων ειναι και ενα θεωρητικο συνολο
